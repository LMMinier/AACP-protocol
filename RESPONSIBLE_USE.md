# Responsible Use and Ethics Policy

Allowed use: prompt-injection resistance research, safer AI-agent runtime design, tool-sink mediation, auditability, governance, and defensive evaluation.

Disallowed use: attacks against third-party systems, bypassing safety systems, malware deployment, credential theft, exfiltration, unauthorized tool execution, and evasion of monitoring.

The project should never ship reusable malicious prompt payloads. Use sanitized markers and abstract labels for tests.
